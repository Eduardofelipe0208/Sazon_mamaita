## ¿Cómo Funciona el Sistema de Reservas?

### 1️⃣ El Cliente Hace una Reserva en el Sitio Web

El cliente ingresa a la sección **"Reservas"** y completa un sencillo formulario:

```
📝 Información Requerida:
- Nombre completo
- Teléfono (número donde contactarlo)
- Fecha que desea venir
- Hora de la reserva
- Cantidad de personas
- Notas especiales (cumpleaños, alergias, etc.)
```

### 2️⃣ El Sistema Envía los Datos a WhatsApp

Una vez que hace clic en **"Confirmar Reserva"**:
- ✅ Se abre automáticamente WhatsApp
- ✅ El mensaje contiene todos los datos de la reserva
- ✅ Solo necesita hacer clic para enviar

### 3️⃣ El Dueño Recibe la Reserva en WhatsApp

El dueño recibe un mensaje profesional como este:

```
NUEVA RESERVA

Nombre: Patricia García
Fecha: 2026-03-15
Hora: 19:00
Personas: 6
Telefono: +58 412 1234567
Notas: Es una celebración de cumpleaños

Enviado desde La Sazon de Mamaiita
```

### 4️⃣ El Dueño Confirma la Reserva

El dueño contesta en WhatsApp confirmando disponibilidad, recomendaciones, etc.
